# Completed PRD Archive

This directory contains archived PRDs that have been fully completed (all features passing).

## How to Use

**Archive a completed PRD:**
```bash
./.claude/ralph/scripts/archive-prd.sh prds/<prd-name>.json
```

**Requirements for archiving:**
- All features must have `passes: true`
- Cannot be `current.json` (active loop protection)
