# kNearme Content Editorial Workflow

## Overview
This document outlines the standard workflow for planning, creating, reviewing, publishing, and measuring content for the kNearme platform. Following this workflow ensures content quality, consistency, and effectiveness in meeting our strategic goals.

## Content Types & Purpose
- **Service Pages**: Convert visitors looking for specific services
- **Educational Content**: Build authority and drive organic traffic
- **Problem-Solution Content**: Address specific customer pain points
- **Project Showcases**: Demonstrate contractor quality and build trust
- **Contractor Features**: Promote contractor partners and their work
- **Evergreen Guides**: Provide ongoing value and establish authority

## Content Planning Workflow

### 1. Research & Topic Selection (Week 1)
#### Responsible: Content Strategist
- [ ] Conduct keyword research for target service category
- [ ] Analyze competitor content and identify gaps
- [ ] Review existing content performance
- [ ] Identify seasonal opportunities
- [ ] Draft monthly content calendar
- [ ] Get approval from Content Manager

### 2. Content Brief Creation (Week 1-2)
#### Responsible: Content Strategist
- [ ] Create detailed content brief using template
- [ ] Define target keywords and search intent
- [ ] Outline content structure and key points
- [ ] Specify word count and visual requirements
- [ ] Identify internal/external linking opportunities
- [ ] Set target metrics for content performance

### 3. Content Production (Week 2-3)
#### Responsible: Content Writer
- [ ] Research topic thoroughly
- [ ] Draft content following brief guidelines
- [ ] Include all required sections
- [ ] Add suggestions for visuals/graphics
- [ ] Optimize for SEO and readability
- [ ] Create meta title and description
- [ ] Submit first draft for review

### 4. Content Review (Week 3)
#### Responsible: Content Editor
- [ ] Review for accuracy and completeness
- [ ] Check SEO optimization and keyword usage
- [ ] Verify tone and brand voice consistency
- [ ] Edit for grammar, spelling, and clarity
- [ ] Ensure adherence to style guide
- [ ] Add suggestions for improvements
- [ ] Return to writer for revisions if needed

### 5. Final Approval (Week 3-4)
#### Responsible: Content Manager
- [ ] Final review of content
- [ ] Approve visual elements and graphics
- [ ] Verify all links and citations
- [ ] Ensure compliance with guidelines
- [ ] Give final approval or request changes

### 6. Visual Production (Parallel to Week 3-4)
#### Responsible: Visual Designer
- [ ] Create/source images based on content requirements
- [ ] Design graphics and diagrams
- [ ] Edit photos as needed
- [ ] Optimize all visual assets for web
- [ ] Add proper alt text and captions
- [ ] Submit visuals for approval

### 7. Content Preparation & Publishing (Week 4)
#### Responsible: Web Content Specialist
- [ ] Format content in CMS
- [ ] Add approved visuals and media
- [ ] Set up all meta information
- [ ] Implement structured data if applicable
- [ ] Schedule publication date
- [ ] Perform final QA check
- [ ] Publish content or queue for scheduled release

### 8. Promotion & Distribution (Week 4-5)
#### Responsible: Content Marketing Specialist
- [ ] Announce on social media channels
- [ ] Include in relevant newsletters
- [ ] Share with featured contractors
- [ ] Add to content distribution plan
- [ ] Set up any paid promotion
- [ ] Monitor initial performance

### 9. Performance Tracking (Ongoing)
#### Responsible: Content Analyst
- [ ] Track content performance metrics
- [ ] Monitor organic traffic and rankings
- [ ] Track conversion metrics
- [ ] Document user engagement
- [ ] Prepare monthly performance report
- [ ] Make optimization recommendations

### 10. Content Optimization (Monthly)
#### Responsible: Content Strategist & SEO Specialist
- [ ] Review performance data
- [ ] Identify optimization opportunities
- [ ] Update underperforming content
- [ ] Enhance high-performing content
- [ ] Implement recommended improvements
- [ ] Schedule follow-up measurement

## Content Quality Standards

### Accuracy & Authority
- All facts must be verified from reliable sources
- Statistics should be current and cited
- Technical information must be accurate and complete
- Expert input should be included when applicable

### SEO Requirements
- Primary keyword in title, H1, and first paragraph
- Secondary keywords distributed naturally
- Proper heading structure (H1, H2, H3, etc.)
- Optimized meta title and description
- Alt text for all images
- Internal links to relevant content
- External links to authoritative sources

### Readability Standards
- Flesch-Kincaid reading level: 8th-10th grade
- Paragraphs limited to 3-4 sentences
- Use subheadings to break up text
- Include bullet points for lists
- Bold important information
- Use plain language when possible
- Avoid industry jargon unless explained

### Visual Requirements
- High-quality, relevant images
- Before/after photos for project content
- Diagrams or infographics for complex concepts
- Consistent branding and style
- Proper attribution for all visuals
- Mobile-friendly design

## Content Governance

### Review Cycles
- Service Pages: Review quarterly
- Educational Content: Review semi-annually
- Project Showcases: Review annually
- Evergreen Guides: Review annually

### Content Maintenance
- Update statistics and facts annually
- Refresh seasonal content 30 days before relevant season
- Update service pages when offerings change
- Add new project showcases as they become available
- Remove outdated information immediately

### Documentation Requirements
- All content should have a completed brief
- Track all content in content inventory spreadsheet
- Document all content changes and updates
- Maintain version history for major content
- Record performance metrics for all content

## Roles & Responsibilities
- **Content Manager**: Overall strategy and final approval
- **Content Strategist**: Planning, briefs, and optimization
- **Content Writer**: Research and content creation
- **Content Editor**: Review, editing, and quality control
- **Visual Designer**: Images, graphics, and visual assets
- **Web Content Specialist**: CMS management and publishing
- **Content Marketing Specialist**: Promotion and distribution
- **Content Analyst**: Measurement and reporting

## Timeline & Deadlines
- Content Calendar: Finalize by 15th of previous month
- Content Briefs: Complete 3 weeks before publication
- First Draft: Submit 2 weeks before publication
- Edits & Revisions: Complete 1 week before publication
- Final Approval: 3 days before publication
- Publication: According to content calendar
- Promotion: Begins on publication day
- Analysis: 30 days after publication
